## Overview

Pictet technical test developed by Miguel Castro.

## Execution

To easily run and test the REST API, simply execute the command <code>docker-compose up --build</code> (with Docker installed)
or with JDK 17 installed, simply run the command <code>mvn clean install</code> followed by <code>mvn spring-boot:run</code>

Once the application is running, simply visit http://localhost:8080/swagger-ui/index.html to explore the application's endpoints.
